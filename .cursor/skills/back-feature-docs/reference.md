### 全局守卫顺序（本项目共用）

与权限、登录文档保持一致：

```
1. JwtAuthGuard      → 401；@Public() 跳过
2. AppThrottlerGuard → 429
3. PermissionsGuard  → 403；@SkipPermissions() / @Public() 跳过
```

ValidationPipe 在 Guard 之后、Controller 之前（`main.ts` 全局注册）。
